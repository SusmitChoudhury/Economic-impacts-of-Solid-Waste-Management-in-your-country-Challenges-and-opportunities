# 🌿 Green India Initiative - Solid Waste Management Awareness

### **Topic: Economic Impacts of Solid Waste Management in India: Challenges and Solutions**

An interactive, academic-quality web platform designed to raise awareness about the environmental and economic crisis of waste management in India. This project gamifies learning and encourages citizen participation through interactive maps, games, and a digital certification system.

🔗 **[Live Demo: View the Website Here](https://SusmitChoudhury.github.io/Economic-impacts-of-Solid-Waste-Management-in-your-country-Challenges-and-opportunities/EcoWasteProject/)**

---

## ✨ Key Features

### 📊 Interactive Data Visualization
* **SVG India Heatmap:** A fully interactive, color-coded map displaying waste generation intensity across North, South, East, West, and Central India.
* **Data Insights:** Dynamic cards explaining the economic risks associated with Critical (Red) to Low (Green) waste zones.

### 🎮 Gamified Learning
* **Waste Segregation Challenge:** A drag-and-drop interactive game teaching users how to correctly sort Wet, Dry, Plastic, and Hazardous waste into specific dustbins.
* **Clean the Area Mission:** A time-based clicker game where users must "clean" a virtual environment within 60 seconds to win.

### 🏆 Citizen Engagement
* **Digital Certificate Generator:** A custom HTML5 Canvas feature that allows users to pledge as a "Green India Volunteer" and download a high-resolution, personalized certificate.
* **Government Schemes:** Information cards on policies like *Swachh Bharat Abhiyan* and *EPR* with direct links to official government portals.

### 💻 Technical Highlights
* **Responsive Design:** Optimized layout for Mobile, Tablet, and Desktop screens.
* **Smooth Animations:** Scroll-triggered reveal effects, parallax hero sections, and ripple buttons.
* **Zero Dependencies:** Built entirely with Vanilla JavaScript, HTML5, and CSS3 for maximum performance.

---

## 🛠️ Tech Stack

* **HTML5:** Semantic structure, SVG Map implementation.
* **CSS3:** Flexbox/Grid layouts, Glassmorphism effects, CSS Variables, and Media Queries.
* **JavaScript (ES6+):** Game logic, DOM manipulation, Intersection Observers, and Canvas API.
